README.md

CASM

This package contains the Matlab implementation of the Context-aware Structural Model in “Context-Aware Activity Recognition and Anomaly Detection in Video”, J-STSP, 2013. and "Context-Aware Modeling and Recognition of Activities in Video", CVPR, 2013.

======================= Dependence =======================

LIBQP: http://ida.first.fraunhofer.de/~franc/libqp/libqp.zip

======================= Main Folders =======================

interface: contains interface functions that need to be defined for particular applications.

demo1: one implementation for context-aware activity recognition, please see the readme file in ./demo1 for details.

demo2: one implementation for context-aware activity recognition, please see the readme file in ./demo2 for details.
